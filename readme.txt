Version Diff helps you see what's changing before you update.

== Description ==

Version Diff allows you to preview update changes before installing them.

When an update is available, Version Diff compares the currently installed version with the update package and presents the differences in an easy-to-navigate interface. Review modified files, inspect added or removed files, and examine code changes before applying an update.

Whether you're a developer, agency, site administrator, or security-conscious user, Version Diff gives you greater visibility into what an update contains.

== Key Features ==

* Preview update changes before installation
* Compare installed files with the available update package
* Interactive file tree explorer
* View added, modified, and deleted files
* Side-by-side code comparison
* File change statistics
* IDE-style review interface
* Works directly from the WordPress Plugins screen
* No external services required

== Why Use Version Diff? ==

Updates can introduce new features, bug fixes, security patches, or significant code changes. Version Diff helps you understand exactly what is changing before updating, allowing you to review updates with confidence.

== How It Works ==

1. Navigate to the Plugins screen.
2. Locate an item with an available update.
3. Click the Version Diff link.
4. Review added, modified, and deleted files.
5. Inspect code differences before deciding to update.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/version-diff` directory, or install through the WordPress Plugins screen.
2. Activate the plugin through the Plugins screen in WordPress.
3. Go to **Plugins**.
4. When an update is available, click **Version Diff** to preview the changes.

== Frequently Asked Questions ==

= Does Version Diff modify any files? =

No. Version Diff only downloads the update package temporarily for comparison purposes and does not modify installed files.

= Does Version Diff perform updates? =

No. The plugin only allows you to inspect changes before updating.

= Can I review code changes line by line? =

Yes. Version Diff provides a side-by-side comparison view for changed files.

= Does it work with WordPress.org updates? =

Yes. Version Diff works with updates delivered through the standard WordPress update system.
